VNgen Console Commands:

(Commands are case-sensitive)

Command: Description: Notes:
exit Close the console
help Display this dialog
game_end() Exits the game
game_restart() Restart the game
room_goto(room) Jump to a specific room (Same as vngen_room_goto)
show_debug_helpers(true/false) Enable or disable wireframe overlays Affects performance!
show_debug_overlay(true/false) Enable or disable live performance info
show_debug_stats(true/false) Enable or disable live engine info Affects performance!
vngen_file_save(filename, [encrypt]) Save VNgen data to a file
vngen_file_load(filename) Load previously saved VNgen data
vngen_file_delete(filename) Delete previously saved VNgen data
vngen_goto(event, [object, perform]) Jump to a specific engine event
vngen_goto_unread([perform]) Jump to the nearest option or unread event
vngen_log_clear([destroy]) Clear the backlog and optionally destroy the log object
vngen_room_goto(room, [event, object], [perform]) Jump to a specific room and optional event
vngen_set_lang(lang, [type]) Set the active language flag for text, audio, or both
vngen_get_lang(type) Display the active language flag for text or audio
vngen_set_renderlevel(level) Set renderlevel, where 0 is default
vngen_set_scale(view) Set a viewport to scale to window dimensions
vngen_set_speed(speed) Set the text typewriter effect speed
vngen_set_vol(type, vol, [fade]) Set the global volume offset for a particular type of sound Use keyword 'all' for all types
vngen_version() Display the current VNgen version and build date
window_set_fullscreen(full) Set window display mode
 
Additional Controls: Description: Notes:
up arrow Previous command Max 32 stored commands
down arrow Next command Max 32 stored commands
enter Execute command
escape Close console
